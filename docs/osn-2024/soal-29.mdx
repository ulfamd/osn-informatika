# Soal 29 — Start dan Finish?

## Soal

Perhatikan fungsi berikut:

```cpp
int go(int N) {
    int s = 0;
    int f = 1;

    for (int i = 2; i <= N; i++) {
        if (N % i == 0) {

            if (i - f > s) {
                s = i - f;
            }

            f = i;
        }
    }

    return s;
}
```

Pertanyaannya:

> Berapakah nilai dari:

```cpp
go(30)
```

Tuliskan jawaban dalam bentuk ANGKA.

---

## Cara Berpikir

Fungsi ini:
- mencari semua faktor dari N
- menghitung selisih antar faktor berurutan
- menyimpan selisih terbesar

Variabel:
- `f` = faktor sebelumnya
- `s` = selisih terbesar sementara

---

Faktor-faktor 30:

```txt
1, 2, 3, 5, 6, 10, 15, 30
```

## Catatan Penting OSN

Soal ini melatih:
- simulasi kode
- faktor bilangan
- tracking maksimum

Konsep fungsi:

```txt
go(N)
=
selisih terbesar antar faktor berurutan N
```

Teknik cepat:
1. Cari semua faktor
2. Urutkan
3. Cari gap terbesar
