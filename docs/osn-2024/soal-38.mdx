# Soal 38 — Fungsi Pendek

## Soal

Perhatikan fungsi berikut:

```cpp
int pndk(int N, int K) {

    if (N == 1)
        return 1;

    else
        return (pndk(N - 1, K) + K - 1) % N + 1;
}
```

Pertanyaannya:

> Berapakah hasil dari:

```cpp
pndk(7, 7)
```

Tuliskan jawaban dalam bentuk ANGKA.

---

## Cara Berpikir

Fungsi ini adalah:

### Josephus Problem

Konsepnya:
- orang berdiri melingkar
- setiap hitungan ke-K dieliminasi
- cari posisi pemenang terakhir

---

## Rumus Rekursif

```cpp
pndk(N,K)
=
(pndk(N-1,K)+K-1)%N + 1
```

Basis:
```cpp
pndk(1,K)=1
```

Kita hitung dari bawah.

- rekursi
- modulo

