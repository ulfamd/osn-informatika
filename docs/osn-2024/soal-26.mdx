# Soal 26 — Cek dan Ricek

## Soal

Perhatikan fungsi berikut:

```cpp
bool cek(string S) {
    int n = S.length();

    for (int i = 0; i < n-1; i++) {
        if (S[i] > S[i+1]) return false;
    }

    return true;
}
```

Dari pilihan berikut, manakah yang menghasilkan:

```txt
TRUE
```

### Pilihan

A. `cek("BEGINI")`

B. `cek("BEGITU")`

C. `cek("BERAPA")`

D. `cek("BERUPA")`

E. `cek("BETAPA")`

---

## Cara Berpikir

Fungsi `cek()` memeriksa apakah:

```txt
huruf-huruf pada string selalu
tidak menurun
```

Artinya:

```txt
S[i] <= S[i+1]
```

untuk semua posisi.

Jika ada:
```txt
S[i] > S[i+1]
```

langsung:
```txt
false
```

---

## Artinya Apa?

String harus urut alfabet naik atau tetap.

Contoh valid:
```txt
AABCD
```

Contoh tidak valid:
```txt
BAC
```

karena:
```txt
B > A
```
