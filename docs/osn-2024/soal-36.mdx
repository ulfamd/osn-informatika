# Soal 36 — Panas Dingin

## Soal

Perhatikan fungsi berikut:

```cpp
int panas(int X) {

    if (X == 0) return 0;

    else return (X % 10) + panas(X / 10);
}
```

```cpp
int dingin(int X, int Y) {

    int air = 0;

    while (panas(air) != X)
        air = air + Y;

    return air;
}
```

Pertanyaannya:

> Berapakah hasil dari:

```cpp
dingin(2, 35)
```

Tuliskan jawaban dalam bentuk ANGKA.

---

## Cara Berpikir

---

## Fungsi panas(X)

Fungsi:

```cpp
panas(X)
```

menghitung:

```txt
jumlah digit dari X
```

Contoh:

```txt
panas(35)
=
3 + 5
=
8
```

---

## Fungsi dingin(X,Y)

Fungsi:
- mulai dari `air = 0`
- terus menambah `Y`
- sampai:

```txt
panas(air) == X
```

---

- digit terakhir selalu 0 atau 5

- kondisi while tidak pernah berhenti

# Jawaban

Soal ini melatih:
- simulasi program
- analisis loop
- observasi matematika

Teknik penting:
- tidak semua loop pasti berhenti
- cek apakah target kondisi mungkin tercapai

Pada soal ini:
```txt
tidak ada kelipatan 35
dengan jumlah digit = 2
```

maka:
```txt
while berjalan selamanya
```
