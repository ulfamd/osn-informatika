# Soal 34 — Dua String

## Soal

Diberikan fungsi berikut:

```cpp
int DUA(string S, string T) {
    int N = S.length();
    int M = T.length();

    int P = 0;

    for (int i = 0; i <= N - M; i++) {

        int Q = 1;

        for (int j = 0; j < M; j++) {
            if (S[i + j] != T[M - 1 - j]) {
                Q = 0;
            }
        }

        P += Q;
    }

    return P;
}
```

Jika:

```txt
S = "ACCAAACCA"
```

berapakah banyak kemungkinan string `T`
sehingga:

```txt
DUA(S,T) = 2
```

Perhatikan:
- tidak ada syarat mengenai panjang `T`.

Tuliskan jawaban dalam bentuk ANGKA.

## Cara Berpikir

Fungsi:

```txt
DUA(S,T)
```

menghitung:
```txt
berapa kali reverse(T)
muncul di S
```

Jadi:
```txt
DUA(S,T)=2
```

berarti:
```txt
reverse(T)
muncul tepat 2 kali di S.
```

Karena:
- operasi reverse bersifat satu-satu,
- jumlah kemungkinan `T`
  sama dengan:
  
```txt
jumlah substring S
yang muncul tepat 2 kali.
```
