# Soal 20 — Kartu Huruf Ajaib

## Soal

Pada awalnya:
- kartu pertama berisi `A`
- kartu kedua berisi `B`
- ...
- kartu ke-26 berisi `Z`

Pak Dengklek menjentikkan jari sebanyak:

```txt
33 kali
```

Perubahan huruf mengikuti tabel pada soal.

Pertanyaannya:

> Apa isi kartu ajaib ke-10 pada akhirnya?

Tuliskan jawaban dalam bentuk HURUF KAPITAL.



---

## Cara Berpikir

Kartu ke-10 awalnya berisi:

```txt
J
```

Dari tabel perubahan:

| Sebelum | Sesudah |
| --- | --- |
| J | E |
| E | H |
| H | F |
| F | I |
| I | E |

Terlihat terbentuk siklus:

```txt
E → H → F → I → E
```

Panjang siklus:

```txt
4
```

---

# Langkah Perubahan

Awal:
```txt
J
```

Setelah 1 kali:
```txt
E
```

Berikutnya mengikuti siklus 4.

Karena total:
```txt
33 kali
```

maka setelah perubahan pertama,
tersisa:

```txt
32 langkah
```

Hitung:

:contentReference[oaicite:1]{index=1}

Hasil:

```txt
0
```

Artinya:
- kembali ke posisi awal siklus.

---

# Posisi Siklus

Siklus dimulai dari:

```txt
E
```

Karena sisa:
```txt
0
```

maka hasil akhirnya:

```txt
E
```

---

# Jawaban

```txt
E
```
