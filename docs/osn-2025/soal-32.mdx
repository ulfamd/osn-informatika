# Soal 32 — Dua String

## Soal

Diberikan fungsi berikut:

```cpp
int DUA(string S, string T) {
    int N = S.length();
    int M = T.length();

    int P = 0;

    for (int i = 0; i <= N - M; i++) {

        int Q = 1;

        for (int j = 0; j < M; j++) {
            if (S[i + j] != T[M - 1 - j]) {
                Q = 0;
            }
        }

        P += Q;
    }

    return P;
}
```

Berapakah hasil dari:

```cpp
DUA("ABCBAABCCBAABC", "CBA")
```

Tuliskan jawaban dalam bentuk ANGKA.

## Cara Berpikir

Fungsi ini membandingkan:

```txt
substring S
```

dengan:

```txt
reverse(T)
```

---

# Langkah 1 — Cari Reverse(T)

Diketahui:

```txt
T = "CBA"
```

Maka:

```txt
reverse(T) = "ABC"
```

---

# Langkah 2 — Cari "ABC" di S

String:

```txt
S = "ABCBAABCCBAABC"
```

Cari seluruh substring:

```txt
"ABC"
```

# Mengapa?

Pada kode:

```cpp
if (S[i + j] != T[M - 1 - j])
```

artinya:
- karakter pada substring `S`
- dibandingkan dengan
  karakter `T` dari belakang.

Jadi sebenarnya:
```txt
mencari reverse(T)
```

## Catatan Penting OSN

Soal ini melatih:
- string matching
- reverse string
- simulasi nested loop

Konsep penting:

Kode:

```cpp
T[M - 1 - j]
```

berarti:
```txt
akses karakter T
dari belakang ke depan.
```

Sehingga fungsi:
```txt
menghitung banyak kemunculan
reverse(T) di S.
```
