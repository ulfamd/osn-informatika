# Soal 32 — Jalur Pendakian

## Soal

Terdapat peta jalur pendakian berupa graph.

Setiap titik:
- menyatakan pos pendakian

Setiap garis:
- menyatakan jalur dua arah

Pendaki ingin berjalan:
- dari Pos A
- kembali lagi ke Pos A
- dan setiap jalur hanya boleh dilewati tepat satu kali

Pertanyaannya:

> Apakah perjalanan tersebut mungkin dilakukan?

Tuliskan jawaban dalam bentuk:

```txt
YA
```

atau

```txt
TIDAK
```
