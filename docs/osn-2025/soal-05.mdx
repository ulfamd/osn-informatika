# Soal 5 — Agen BB007

## Soal

Sebuah nama samaran valid harus memenuhi minimal satu aturan berikut:

---

## Aturan 1

Nama terdiri dari:
- 0 hingga 2 huruf kapital (`A-Z`)
- diikuti 1 hingga 3 digit (`0-9`)

Contoh:
```txt
BB007
K3
47
```

---

## Aturan 2

Nama terdiri dari:
- 1 hingga 3 digit (`0-9`)
- diikuti 0 hingga 2 huruf kapital (`A-Z`)

Contoh:
```txt
007BB
47
```

---

Pertanyaannya:

> Berapa banyak nama samaran berbeda yang valid?

Tuliskan jawaban dalam bentuk ANGKA.

---

## Cara Berpikir

Kita hitung:
- semua string valid aturan 1
- semua string valid aturan 2
- lalu kurangi yang terhitung dua kali

---

# Langkah 1 — Hitung Aturan 1

Format:

```txt
(huruf 0-2) + (digit 1-3)
```

---

## Huruf 0 karakter

Pilihan:
```txt
1
```

---

## Huruf 1 karakter

```txt
26
```

---

## Huruf 2 karakter

```txt
26 × 26
=
676
```

---

## Total Bagian Huruf

```txt
1 + 26 + 676
=
703
```

---

## Digit 1 karakter

```txt
10
```

---

## Digit 2 karakter

```txt
10²
=
100
```

---

## Digit 3 karakter

```txt
10³
=
1000
```

---

## Total Bagian Digit

```txt
10 + 100 + 1000
=
1110
```

---

## Total Aturan 1

```txt
703 × 1110
=
780330
```

---

# Langkah 2 — Hitung Aturan 2

Strukturnya simetris:
- digit dulu
- huruf kemudian

Jumlahnya sama:

```txt
780330
```

---

# Langkah 3 — Cari Yang Terhitung Dua Kali

String yang memenuhi:
- aturan 1
- dan aturan 2

harus:
- hanya digit saja

Karena:
- aturan 1 mengharuskan digit di belakang
- aturan 2 mengharuskan digit di depan

Agar sama:
```txt
tidak boleh ada huruf
```

---

## Banyak String Digit Saja

Digit:
- panjang 1 sampai 3

Jumlah:

```txt
10 + 100 + 1000
=
1110
```

---

# Langkah 4 — Inclusion Exclusion

```txt
780330 + 780330 - 1110
```

Hitung:

```txt
1560660 - 1110
=
1559550
```

---

# Jawaban

1559550

---

## Catatan Penting OSN

Soal ini melatih:
- counting
- kombinatorika
- inclusion exclusion principle

Konsep penting:

```txt
Total(A ∪ B)
=
A + B - irisan
```

Jebakan utama:
- string digit murni
- dihitung dua kali.
