# Soal 10 — Dua dari Tujuh Telur Bebek

## Soal

Pak Dengklek memiliki tujuh buah telur bebek yang diberi label:

```txt
1 sampai 7
```

Diketahui:
- tepat dua telur adalah telur emas,
- tetapi semua telur terlihat sama.

Pak Dengklek memiliki mesin pendeteksi telur emas.

Cara kerja mesin:
1. Pak Dengklek memasukkan beberapa telur.
2. Mesin mengembalikan semua telur.
3. Mesin hanya memberikan salah satu jawaban:
   - `"TIDAK ADA TELUR EMAS"`
   - `"TELUR EMAS TERDETEKSI"`

Jika mesin mengatakan:
```txt
TELUR EMAS TERDETEKSI
```

artinya:
- minimal ada satu telur emas di dalam kumpulan tersebut.

Pak Dengklek ingin:
- menemukan tepat dua telur emas
- dengan jumlah penggunaan mesin seminimum mungkin
- untuk kasus terburuk.

Pertanyaannya:

> Berapakah nilai minimum K sehingga
> Pak Dengklek selalu dapat menentukan
> kedua telur emas dengan paling banyak K pengecekan?

Tuliskan jawaban dalam bentuk ANGKA.

:contentReference[oaicite:0]{index=0}

---

## Cara Berpikir

---

# Langkah 1 — Banyak Kemungkinan

Karena:
- ada 7 telur
- pilih 2 telur emas

Jumlah kemungkinan:

:contentReference[oaicite:1]{index=1}

Hitung:

```txt
21 kemungkinan
```

---

# Langkah 2 — Informasi dari Mesin

Setiap pengecekan:
- hanya menghasilkan 2 kemungkinan jawaban

yaitu:
```txt
YA / TIDAK
```

Artinya:
- satu pengecekan maksimal memberi:
```txt
1 bit informasi
```

---

# Langkah 3 — Cari Minimum Pengecekan

Kita perlu membedakan:
```txt
21 kemungkinan
```

Dengan pertanyaan biner.

Cari:
```txt
2^K ≥ 21
```

---

# Hitung

```txt
2^4 = 16
```

masih kurang.

```txt
2^5 = 32
```

cukup.

---

# Jadi

Minimum:

```txt
K = 5
```

---

# Jawaban

5

---

## Catatan Penting OSN

Soal ini melatih:
- information theory
- binary decision tree
- lower bound

Konsep penting:

Jika:
- ada N kemungkinan,
- dan tiap pertanyaan hanya punya 2 jawaban,

maka minimum pertanyaan:

:contentReference[oaicite:2]{index=2}

Pada soal ini:

```txt
N = 21
```

maka:
```txt
ceil(log₂21) = 5
```
