# Soal 10 — Rangkaian Mantra

## Soal

Terdapat 5 kartu:

```txt
[v, w, x, y, z]
```

Awalnya berisi:

```txt
[1, 2, 3, 4, 5]
```

---

## Mantra A

```txt
[v,w,x,y,z]
→
[x,w,v,y,z]
```

---

## Mantra B

```txt
[v,w,x,y,z]
→
[y,v,w,z,x]
```

---

## Mantra C

```txt
[v,w,x,y,z]
→
[w,x,y,z,v]
```

Pertanyaannya:

> Berapa nilai minimum X sehingga:

```txt
F(B,X)
```

menghasilkan kembali urutan awal?

Tuliskan jawaban dalam bentuk ANGKA.

---

## Cara Berpikir

Kita cari:
- berapa kali mantra B
- harus dilakukan
- agar kembali ke posisi awal

Ini adalah:

```txt
siklus permutasi
```

---

# Analisis Mantra B

Mantra B:

```txt
[v,w,x,y,z]
→
[y,v,w,z,x]
```

Mari lihat perpindahan posisi:

| Posisi Lama | Pindah ke |
| --- | --- |
| v | posisi 2 |
| w | posisi 3 |
| x | posisi 5 |
| y | posisi 1 |
| z | posisi 4 |

---

# Simulasi

Awal:

```txt
[1,2,3,4,5]
```

---

## B sekali

```txt
[4,1,2,5,3]
```

---

## B dua kali

Terapkan lagi:

```txt
[5,4,1,3,2]
```

---

## B tiga kali

```txt
[3,5,4,2,1]
```

---

## B empat kali

```txt
[2,3,5,1,4]
```

---

## B lima kali

```txt
[1,2,3,4,5]
```

Kembali ke awal.

---

# Jawaban

5

---

## Catatan Penting OSN

Soal ini melatih:
- permutasi
- siklus transformasi
- simulasi array

Konsep penting:

Sebuah permutasi:
```txt
akan kembali ke bentuk awal
setelah sejumlah langkah tertentu
```

Jumlah langkah minimum disebut:

```txt
order permutasi
```

Pada mantra B:
- seluruh elemen membentuk satu siklus panjang 5.
