# Soal 37 — Rahasia Petak 5x5

## Soal

Diberikan fungsi berikut:

```cpp
int RAHASIA(vector<int> A) {
    int N = A.size();
    int sum = 0;

    for (int i = 0; i < N; i++) {

        int ctr = 0;

        for (int j = 0; j < N; j++) {

            if (A[j] > A[i]) {
                ctr += 1;
            } else if (A[j] < A[i]) {
                ctr -= 1;
            }

        }

        if (ctr == 0) {
            sum += A[i];
        }

    }

    return sum;
}

int PETAK(vector<vector<int>> B) {
    int N = B.size();

    vector<int> C;

    for (int i = 0; i < N; i++) {
        C.push_back(RAHASIA(B[i]));
    }

    return RAHASIA(C);
}
```

Akan dipanggil:

```cpp
PETAK(
 {a,b,c,d,e},
 {f,g,h,i,j},
 {k,l,m,n,o},
 {p,q,r,s,t},
 {u,v,w,x,y}
)
```

dengan:
- setiap huruf berisi bilangan berbeda
- dari 1 sampai 25.

Pertanyaannya:

> Berapakah hasil kembalian terbesar
> yang mungkin dihasilkan?

Tuliskan jawaban dalam bentuk ANGKA.



---

## Cara Cepat

Dari soal sebelumnya:

```txt
RAHASIA(A)
=
median(A)
```

Maka:

```txt
PETAK(B)
=
median
dari median tiap baris
```

---

# Tujuan

Kita ingin:
```txt
hasil sebesar mungkin
```

Artinya:
- median dari 5 median baris
  harus sebesar mungkin.

---

# Fakta Penting

Median suatu baris berisi 5 angka adalah:
```txt
angka ke-3 terbesar
```

Agar median besar:
- minimal harus ada 3 angka besar
  di baris tersebut.

---

# Strategi Maksimum

Kita ingin median akhir sebesar mungkin.

Karena:
```txt
median dari 5 median
```

maka:
- minimal 3 baris
  harus memiliki median besar.

---

# Gunakan Angka Terbesar

Ambil 15 angka terbesar:

```txt
11 sampai 25
```

Karena:
- untuk membuat 3 median besar,
- dibutuhkan minimal:
  
```txt
3 × 3 = 9 angka besar
```

---

# Median Maksimum yang Bisa Dijamin

Kita ingin:
```txt
minimal 3 baris
memiliki median ≥ X
```

Berarti:
- tiap baris butuh minimal 3 angka ≥ X.

Total:
```txt
9 angka ≥ X
```

---

# Cari X Terbesar

Bilangan:
```txt
17 sampai 25
```

jumlahnya:
```txt
9 angka
```

Pas.

Artinya:
- kita bisa membuat
  3 baris dengan median minimal:

```txt
17
```

Tetapi:
```txt
18 sampai 25
```

hanya:
```txt
8 angka
```

tidak cukup.

---

# Maka

Median terbesar yang mungkin:

```txt
17
```

---

# Jawaban

17

---

## Catatan Penting OSN

- median

Konsep penting:

```txt
median dari median
```

dimaksimalkan dengan:
- menyebarkan angka besar
  agar minimal 3 grup
  memiliki median besar.
