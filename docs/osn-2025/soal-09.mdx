# Soal 9 — Rangkaian Mantra

## Soal

Terdapat 5 kartu:

```txt
[v, w, x, y, z]
```

Awalnya berisi:

```txt
[1, 2, 3, 4, 5]
```

---

## Mantra A

```txt
[v,w,x,y,z]
→
[x,w,v,y,z]
```

---

## Mantra B

```txt
[v,w,x,y,z]
→
[y,v,w,z,x]
```

---

## Mantra C

```txt
[v,w,x,y,z]
→
[w,x,y,z,v]
```

Pertanyaannya:

> Berapa nilai minimum X sehingga:

```txt
F(A,X)
```

menghasilkan kembali urutan awal?

Tuliskan jawaban dalam bentuk ANGKA.

---

## Cara Berpikir

Kita cari:

```txt
berapa kali mantra A
harus dilakukan
agar kembali ke semula
```

Ini disebut:

```txt
siklus / order permutasi
```

---

# Analisis Mantra A

Mantra A:

```txt
[v,w,x,y,z]
→
[x,w,v,y,z]
```

Artinya:
- posisi 1 dan 3 ditukar
- posisi lain tetap

---

# Simulasi

Awal:

```txt
[1,2,3,4,5]
```

---

## Setelah A sekali

```txt
[3,2,1,4,5]
```

---

## Setelah A dua kali

Tukar lagi posisi 1 dan 3:

```txt
[1,2,3,4,5]
```

Kembali seperti awal.

---

# Jawaban

2

---

## Catatan Penting OSN

Soal ini melatih:
- permutasi
- transformasi array
- cycle/order

Konsep penting:

Operasi swap:
```txt
diterapkan 2 kali
→ kembali semula
```

Karena:
```txt
swap adalah involution
```

artinya:
```txt
A(A(x)) = x
```
