# Soal 30 — FUN dan GSI

## Soal

Diberikan potongan kode berikut:

```cpp
int GSI(vector<int> B, int x, int y) {
    return B[y] - B[x - 1];
}

int FUN(vector<int> A) {
    int N = A.size();

    vector<int> B;
    B.push_back(0);

    for (int i = 0; i < N; i++) {
        B.push_back(B[i] + A[i]);
    }

    int out = 0;

    for (int i = 1; i <= N; i++) {
        for (int j = i; j <= N; j++) {
            out = max(out, GSI(B, i, j));
        }
    }

    return out;
}
```

Jika:

```txt
A = {1, 2, 3, 4, 5}
```

manakah pemanggilan berikut
yang menghasilkan nilai terbesar?

```txt
A. GSI(B, 1, 4)
B. GSI(B, 2, 4)
C. GSI(B, 4, 5)
D. GSI(B, 5, 5)
E. GSI(B, 3, 4)
```

# Fungsi GSI

Perhatikan:

```cpp
GSI(B,x,y)
=
B[y] - B[x-1]
```

Artinya:
```txt
jumlah elemen array
dari indeks x sampai y
```

# Hitung Cepat

---

```txt
= B[4] - B[0]
```

# Catatan Penting OSN

Soal ini melatih:
- prefix sum
- range sum query
- observasi kode

Konsep penting:

Dengan prefix sum:

```txt
sum(l,r)
=
prefix[r] - prefix[l-1]
```

kita bisa menghitung:
```txt
jumlah subarray
dalam O(1).
```
