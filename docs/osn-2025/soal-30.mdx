# Soal 30 — FUN dan GSI

## Soal

Diberikan potongan program berikut:

```cpp
int GSI(vector<int> B, int x, int y) {
    return B[y] - B[x - 1];
}

int FUN(vector<int> A) {
    int N = A.size();

    vector<int> B;
    B.push_back(0);

    for (int i = 0; i < N; i++) {
        B.push_back(B[i] + A[i]);
    }

    int out = 0;

    for (int i = 1; i <= N; i++) {
        for (int j = i; j <= N; j++) {
            out = max(out, GSI(B, i, j));
        }
    }

    return out;
}
```

Apabila:

```txt
A = {1, 2, 3, 4, 5}
```

maka dari 5 pilihan berikut,
manakah yang hasil kembaliannya paling besar?

# Rumus Fungsi GSI

Fungsi:

```cpp
GSI(B,x,y)
=
B[y] - B[x-1]
```

digunakan untuk menghitung:

```txt
jumlah subarray
dari indeks x sampai y
```

---

# Hitung Satu per Satu

---

## A. GSI(B,1,4)

```txt
= B[4] - B[0]
= 10 - 0
= 10
```

---

## B. GSI(B,2,4)

```txt
= B[4] - B[1]
= 10 - 1
= 9
```

---

## C. GSI(B,4,5)

```txt
= B[5] - B[3]
= 15 - 6
= 9
```

---

## D. GSI(B,5,5)

```txt
= B[5] - B[4]
= 15 - 10
= 5
```

---

## E. GSI(B,3,4)

```txt
= B[4] - B[2]
= 10 - 3
= 7
```
