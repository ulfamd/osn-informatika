# Soal 40 — Binary Search

## Soal

Terdapat array terurut berisi:

```txt
4096 elemen
```

Dilakukan binary search untuk mencari suatu elemen.

Pertanyaannya:

> Setelah 5 kali pemeriksaan,
> paling banyak masih tersisa berapa elemen
> yang mungkin mengandung target?

Tuliskan jawaban dalam bentuk ANGKA.

---

## Cara Berpikir

Binary Search:
- setiap pemeriksaan membagi jumlah data menjadi dua.

---

Jumlah elemen:

