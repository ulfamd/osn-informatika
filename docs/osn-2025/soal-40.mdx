# Soal 40 — Merah-Putih Nusantara

## Soal

Diberikan potongan kode berikut:

```cpp
int MERAH(int A, int B) {
    if (B == 0) {
        return A;
    } else {
        return MERAH(B, A % B);
    }
}

int PUTIH(int A, int B, int C) {
    if (C == 0) {
        return 0;
    } else if (MERAH(A, C) == B) {
        return 1 + PUTIH(A, B, C - 1);
    } else {
        return PUTIH(A, B, C - 1);
    }
}

int NUSANTARA(int A, int B) {
    return PUTIH(A, B, A);
}
```

Berapakah hasil dari:

```cpp
NUSANTARA(2025, 135)
```

Tuliskan jawaban dalam bentuk ANGKA.

---

# Cara Cepat

## Langkah 1 — Pahami Fungsi

Fungsi:

```cpp
MERAH(A,B)
```

menghasilkan:

```txt
FPB(A,B)
```

Sedangkan:

```cpp
NUSANTARA(A,B)
```

menghitung banyak bilangan:

```txt
x
```

dengan:

```txt
1 ≤ x ≤ A
```

yang memenuhi:

```txt
FPB(A,x)=B
```

---

# Jadi

```cpp
NUSANTARA(2025,135)
```

berarti:

```txt
menghitung banyak x
```

dengan:

```txt
FPB(2025,x)=135
```

---

# Langkah 2 — Faktorkan

Karena:

```txt
2025 = 135 × 15
```

maka:

```txt
x harus kelipatan 135
```

Misalkan:

```txt
x = 135k
```

Substitusi:

```txt
FPB(2025,135k)=135
```

Bagi 135:

```txt
FPB(15,k)=1
```

Artinya:
- kita mencari banyak bilangan:
  
```txt
k
```

yang:
```txt
relatif prima dengan 15
```

---

# Langkah 3 — Batas k

Karena:

```txt
x ≤ 2025
```

dan:

```txt
x = 135k
```

maka:

```txt
k ≤ 15
```

Jadi kita cukup cek:

```txt
1 sampai 15
```

---

# Langkah 4 — Cari yang Coprime dengan 15

Bilangan yang relatif prima dengan 15:

```txt
1, 2, 4, 7, 8, 11, 13, 14
```

Jumlah:

```txt
8
```

---

# Jawaban

```txt
8
```

---

# Catatan Penting OSN

Soal ini melatih:
- FPB / GCD
- teori bilangan
- Euler Totient

Konsep penting:

Jika:

```txt
FPB(A,x)=B
```

maka:

```txt
x = Bk
```

dan berubah menjadi:

```txt
FPB(A/B,k)=1
```

yang berarti:
```txt
k harus coprime
dengan A/B.
```
