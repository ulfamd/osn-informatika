# Soal 39 — Merah-Putih Nusantara

## Soal

Diberikan potongan kode berikut:

```cpp
int MERAH(int A, int B) {
    if (B == 0) {
        return A;
    } else {
        return MERAH(B, A % B);
    }
}

int PUTIH(int A, int B, int C) {
    if (C == 0) {
        return 0;
    } else if (MERAH(A, C) == B) {
        return 1 + PUTIH(A, B, C - 1);
    } else {
        return PUTIH(A, B, C - 1);
    }
}

int NUSANTARA(int A, int B) {
    return PUTIH(A, B, A);
}
```

Berapakah hasil dari:

```cpp
NUSANTARA(12, 3)
```

Tuliskan jawaban dalam bentuk ANGKA.

---

# Cara Cepat

## Langkah 1 — Pahami Fungsi MERAH

Fungsi:

```cpp
MERAH(A,B)
```

menghasilkan:

```txt
FPB(A,B)
```

---

## Langkah 2 — Pahami Fungsi PUTIH

Bagian:

```cpp
if (MERAH(A, C) == B)
```

berarti:
- menghitung banyak bilangan:
  
```txt
C
```

yang memenuhi:

```txt
FPB(A,C)=B
```

Karena:

```cpp
PUTIH(A,B,C-1)
```

dipanggil terus sampai:
```txt
C=0
```

maka fungsi menghitung:
```txt
berapa banyak bilangan
1 sampai A
```

yang memiliki FPB terhadap A sebesar B.

---

# Jadi

```cpp
NUSANTARA(12,3)
```

berarti:

```txt
menghitung banyak x
dari 1 sampai 12
```

yang memenuhi:

```txt
FPB(12,x)=3
```

```txt
3 dan 9
```

2

# Jawaban

```txt
2
```

# Catatan Penting OSN

Soal ini melatih:
- FPB / GCD
- rekursi
- counting number theory

Konsep penting:

```cpp
NUSANTARA(A,B)
```

menghitung:

```txt
banyak bilangan x
dengan:
FPB(A,x)=B
```

untuk:

```txt
1 ≤ x ≤ A
```
