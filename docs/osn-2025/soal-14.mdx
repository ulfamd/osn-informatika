# Soal 14 — Jalan Tol Bebek

## Soal

Pak Dengklek ingin pergi dari kota:

```txt
A → H
```

Setiap jalan:
- memiliki biaya
- boleh dilewati berkali-kali

Pertanyaannya:

> Berapa banyak jalur berbeda dari A ke H
> yang memiliki total biaya tepat:

```txt
18
```

Tuliskan jawaban dalam bentuk ANGKA.

---

## Cara Berpikir

Kita cari:
- semua jalur dari A ke H
- dengan total biaya = 18

---

## Catatan Penting OSN

Soal ini melatih:
- graph traversal
- path enumeration
- weighted path counting

Konsep penting:

```txt
Shortest path ≠ jumlah path
```

Kadang:
- hanya ada satu jalur optimal
- kadang banyak jalur dengan biaya sama.

Untuk soal kecil:
- brute force manual masih memungkinkan.
