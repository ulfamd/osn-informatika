# Soal 14 — Perjalanan ke Timur

## Soal

Bebek-bebek Pak Dengklek ingin pergi:
- dari titik A
- menuju titik H

Mereka hanya boleh:
```txt
bergerak ke arah kanan
```

Pada graph terdapat:
- beberapa jalur biasa,
- dan 3 jalur rahasia bergembok.

Untuk Soal 14:
```txt
bebek tidak memiliki kunci sakti
```

Artinya:
- seluruh jalur bergembok tidak bisa digunakan.

Pertanyaannya:

> Berapa banyak kemungkinan rute berbeda
> dari A ke H?

Tuliskan jawaban dalam bentuk ANGKA.



---

## Cara Berpikir

Karena:
```txt
tidak punya kunci
```

maka jalur berikut tidak boleh dipakai:

- E → D
- D → F
- salah satu D → G

---

# Hitung Banyak Jalur

Kita hitung jumlah cara mencapai tiap titik.

---

# Dari A ke B

Pada gambar:
- ada 2 jalur dari A ke B.

Maka:

```txt
B = 2
```

---

# Dari B ke C

Hanya:
```txt
1 jalur
```

Maka:

```txt
C = 2
```

---

# Dari B ke D

Ada:
```txt
2 jalur
```

Maka:

```txt
D = 2 × 2 = 4
```

---

# Dari B ke E

Ada:
```txt
1 jalur
```

Maka:

```txt
E = 2
```

---

# Dari E ke F

Karena:
```txt
D → F terkunci
```

maka hanya:
```txt
E → F
```

Ada:
```txt
2 jalur
```

Maka:

```txt
F = 2 × 2 = 4
```

---

# Menuju G

Dari gambar:
- C → G : 1 jalur
- D → G : tinggal 1 jalur terbuka
- F → G : 1 jalur

Maka:

```txt
G = 2 + 4 + 4
=
10
```

---

# Dari G ke H

Ada:
```txt
2 jalur
```

Maka:

```txt
H = 10 × 2
=
20
```

---

# Jawaban

20

---

## Catatan Penting OSN

Soal ini melatih:
- graph DAG
- counting paths
- dynamic programming pada graph

Konsep penting:

Pada graph tanpa siklus:

```txt
jumlah cara ke suatu titik
=
jumlah seluruh cara
dari titik-titik sebelumnya.
```
