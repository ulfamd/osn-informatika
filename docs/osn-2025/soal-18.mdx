# Soal 18 — Menyimpan Kotak Penyimpanan

## Soal

BENAR atau SALAH:

> Apabila seluruh kotak kardus di gudang
> memiliki panjang sisi yang berbeda-beda,
> maka banyaknya kotak kardus paling sedikit
> yang mungkin terlihat dari luar
> adalah satu kotak saja.

Tuliskan jawaban dalam bentuk:

```txt
BENAR
```

atau

```txt
SALAH
```

---

## Cara Berpikir

Aturan:
- sebuah kotak dapat dimasukkan
  ke dalam kotak lain
- jika ukuran sisi kotak di dalam
  lebih kecil.

Karena:
```txt
seluruh ukuran berbeda
```

maka kita bisa mengurutkan:

```txt
kecil → besar
```

---

# Contoh

Misalnya ukuran:

```txt
1, 2, 3, 4, 5
```

Maka bisa dibuat:

```txt
1 masuk 2
2 masuk 3
3 masuk 4
4 masuk 5
```

Sehingga:
- dari luar hanya terlihat:

```txt
5
```

yaitu:
```txt
1 kotak saja
```

## Catatan Penting OSN

Soal ini melatih:
- greedy
- nesting
- chain construction

Konsep penting:

Jika:
```txt
semua ukuran berbeda
```

maka seluruh kotak dapat dibuat:

```txt
satu rantai penuh
```

sehingga:
```txt
hanya satu kotak
yang terlihat dari luar.
```
