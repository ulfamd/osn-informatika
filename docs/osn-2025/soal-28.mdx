# Soal 28 — Modulo Tiga

## Soal

Diberikan fungsi berikut:

```cpp
int TIGA(int N) {
    if (N <= 1) {
        return 1;
    } else if (N % 3 == 0) {
        return TIGA(N - 1) + TIGA(N - 3);
    } else if (N % 3 == 1) {
        return TIGA(N - 2);
    } else {
        return TIGA(N - 3);
    }
}
```

Pertanyaannya:

> Berapakah hasil dari:
>
> ```txt
> TIGA(1) + TIGA(2) + ... + TIGA(100)
> ```

Tuliskan jawaban dalam bentuk ANGKA.

Kita cari pola nilai:


## TIGA(9)

```txt
= TIGA(8) + TIGA(6)
= 1 + 3
= 4
```

Mulai terlihat pola:

| N | TIGA(N) |
|---|---|
| 3 | 2 |
| 6 | 3 |
| 9 | 4 |

Maka:

Jika:
```txt
N = 3k
```

maka:

```txt
TIGA(N) = k + 1
```

Sedangkan:
```txt
N tidak habis dibagi 3
```

selalu:

```txt
TIGA(N) = 1
```

## Bilangan Kelipatan 3

Kelipatan 3:

```txt
3,6,9,...,99
```

Ada:
```txt
33 bilangan
```

Nilainya:

| N | Nilai |
|---|---|
| 3 | 2 |
| 6 | 3 |
| 9 | 4 |
| ... | ... |
| 99 | 34 |

Jumlah:

```txt
2 + 3 + 4 + ... + 34
```

Gunakan rumus deret:

Jumlah:
```txt
1 + 2 + ... + 34
=
34×35/2
=
595
```

Kurangi 1:

```txt
595 - 1
=
594
```

# Total Akhir

```txt
67 + 594
=
661
```

661
