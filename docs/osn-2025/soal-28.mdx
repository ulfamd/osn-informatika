# Soal 28 — Modulo Tiga

## Soal

Diberikan fungsi berikut:

```cpp
int TIGA(int N) {
    if (N <= 1) {
        return 1;
    } else if (N % 3 == 0) {
        return TIGA(N - 1) + TIGA(N - 3);
    } else if (N % 3 == 1) {
        return TIGA(N - 2);
    } else {
        return TIGA(N - 3);
    }
}
```

Berapakah hasil dari:

```txt
TIGA(1) + TIGA(2) + TIGA(3) + ... + TIGA(100)
```

Tuliskan jawaban dalam bentuk ANGKA.

---

# Cara Cepat

Perhatikan pola nilai:

```txt
TIGA(N)
```

---

# Jika N Tidak Habis Dibagi 3

Dari soal sebelumnya:

```txt
TIGA(N) = 1
```

untuk:

```txt
N % 3 ≠ 0
```

---

# Jika N Habis Dibagi 3

Coba hitung beberapa nilai:

| N | TIGA(N) |
|---|---|
| 3 | 2 |
| 6 | 3 |
| 9 | 4 |
| 12 | 5 |

Terlihat pola:

```txt
TIGA(3k) = k + 1
```

---

# Langkah 1 — Hitung Bilangan yang Bukan Kelipatan 3

Dari:

```txt
1 sampai 100
```

Jumlah kelipatan 3:

```txt
⌊100/3⌋ = 33
```

Maka:
- bukan kelipatan 3:

```txt
100 - 33
=
67
```

Karena masing-masing bernilai:

```txt
1
```

maka total:

```txt
67
```

---

# Langkah 2 — Jumlahkan Kelipatan 3

Kelipatan 3:

```txt
3, 6, 9, ..., 99
```

Ada:

```txt
33 bilangan
```

Nilainya:

| Bilangan | Nilai |
|---|---|
| 3 | 2 |
| 6 | 3 |
| 9 | 4 |
| ... | ... |
| 99 | 34 |

Jadi jumlahnya:

```txt
2 + 3 + 4 + ... + 34
```

Gunakan rumus deret:

```txt
1 + 2 + ... + 34
=
34 × 35 / 2
=
595
```

Kurangi 1:

```txt
595 - 1
=
594
```

---

# Langkah 3 — Total Akhir

```txt
67 + 594
=
661
```

---

# Jawaban

```txt
661
```

---

# Catatan Penting OSN

Soal ini melatih:
- rekursi
- pola bilangan
- deret aritmetika

Konsep penting:

Untuk:

```txt
N = 3k
```

berlaku pola:

```txt
TIGA(3k)=k+1
```

Sedangkan:
- selain kelipatan 3,
- nilainya selalu:

```txt
1
```
