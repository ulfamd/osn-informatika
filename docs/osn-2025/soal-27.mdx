# Soal 27 — Modulo Tiga

## Soal

Diberikan fungsi berikut:

```cpp
int TIGA(int N) {
    if (N <= 1) {
        return 1;
    } else if (N % 3 == 0) {
        return TIGA(N - 1) + TIGA(N - 3);
    } else if (N % 3 == 1) {
        return TIGA(N - 2);
    } else {
        return TIGA(N - 3);
    }
}
```

Pertanyaannya:

> Dari pemanggilan:
>
> ```txt
> TIGA(1), TIGA(2), ..., TIGA(2025)
> ```
>
> berapa banyak yang menghasilkan:
>
> ```txt
> 1
> ```

Tuliskan jawaban dalam bentuk ANGKA.

---

# Cara Cepat

Perhatikan pola berdasarkan:

```txt
N % 3
```

---

# Kasus 1

Jika:

```txt
N % 3 = 1
```

maka:

```cpp
TIGA(N) = TIGA(N-2)
```

Nilai akan terus turun sampai:
```txt
≤ 1
```

Sehingga hasil akhirnya:

```txt
1
```

---

# Kasus 2

Jika:

```txt
N % 3 = 2
```

maka:

```cpp
TIGA(N) = TIGA(N-3)
```

Terus berkurang 3,
hingga akhirnya:

```txt
≤ 1
```

Maka hasilnya:

```txt
1
```

---

# Kasus 3

Jika:

```txt
N % 3 = 0
```

maka:

```cpp
TIGA(N)
=
TIGA(N-1) + TIGA(N-3)
```

Karena:
- kedua nilai minimal 1,
- maka hasil minimal:

```txt
2
```

Jadi:
```txt
tidak mungkin bernilai 1
```

---

# Kesimpulan

```txt
TIGA(N) = 1
```

terjadi ketika:

```txt
N tidak habis dibagi 3
```

---

# Hitung Banyaknya

Dari:

```txt
1 sampai 2025
```

Jumlah kelipatan 3:

:contentReference[oaicite:0]{index=0}

Maka:
- yang bukan kelipatan 3:

```txt
2025 - 675
=
1350
```

---

# Jawaban

```txt
1350
```

---

# Catatan Penting OSN

Soal ini melatih:
- rekursi
- pola modulo
- observasi fungsi

Konsep penting:

Untuk:

```txt
N % 3 ≠ 0
```

fungsi hanya:
```txt
mengurangi N
```

hingga basis.

Sedangkan:

```txt
N % 3 = 0
```

menghasilkan:
```txt
penjumlahan dua nilai
```

sehingga hasilnya selalu lebih dari 1.
