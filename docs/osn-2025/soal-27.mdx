# Soal 27 — Modulo Tiga

## Soal

Diberikan fungsi berikut:

```cpp
int TIGA(int N) {
    if (N <= 1) {
        return 1;
    } else if (N % 3 == 0) {
        return TIGA(N - 1) + TIGA(N - 3);
    } else if (N % 3 == 1) {
        return TIGA(N - 2);
    } else {
        return TIGA(N - 3);
    }
}
```

Pertanyaannya:

> Dari pemanggilan:
>
> ```txt
> TIGA(1), TIGA(2), ..., TIGA(2025)
> ```
>
> berapa banyak yang menghasilkan:
>
> ```txt
> 1
> ```

Tuliskan jawaban dalam bentuk ANGKA.

:contentReference[oaicite:0]{index=0}

---

## Cara Berpikir

Kita cari pola nilai:

```txt
TIGA(N)
```

---

# Basis

Jika:

```txt
N <= 1
```

maka:

```txt
TIGA(N) = 1
```

---

# Analisis Berdasarkan Modulo 3

---

## Kasus 1

Jika:

```txt
N % 3 = 1
```

maka:

```txt
TIGA(N)
=
TIGA(N-2)
```

Nilai akan terus mundur
hingga mencapai:

```txt
1 atau negatif
```

Maka:
```txt
selalu bernilai 1
```

---

## Kasus 2

Jika:

```txt
N % 3 = 2
```

maka:

```txt
TIGA(N)
=
TIGA(N-3)
```

Nilai terus berkurang 3,
dan akhirnya mencapai:

```txt
2 → -1
```

Maka:
```txt
selalu bernilai 1
```

---

## Kasus 3

Jika:

```txt
N % 3 = 0
```

maka:

```txt
TIGA(N)
=
TIGA(N-1) + TIGA(N-3)
```

Karena:
- kedua bagian positif,
- nilainya minimal:

```txt
2
```

Jadi:
```txt
tidak pernah bernilai 1
```

---

# Kesimpulan

```txt
TIGA(N) = 1
```

terjadi untuk:

```txt
N tidak habis dibagi 3
```

---

# Hitung Banyaknya

Dari:
```txt
1 sampai 2025
```

Jumlah bilangan habis dibagi 3:

:contentReference[oaicite:1]{index=1}

Total bilangan:

```txt
2025
```

Maka yang TIDAK habis dibagi 3:

```txt
2025 - 675
=
1350
```

---

# Jawaban

1350

---

## Catatan Penting OSN

Soal ini melatih:
- rekursi
- pola modulo
- observasi matematis

Konsep penting:

Untuk:
```txt
N % 3 ≠ 0
```

fungsi hanya:
```txt
mengurangi N
```

hingga basis,
tanpa penjumlahan.

Sedangkan:
```txt
N % 3 = 0
```

menghasilkan:
```txt
penjumlahan dua nilai
```

sehingga hasilnya lebih besar dari 1.
