# Soal 33 — Dua String

## Soal

Diberikan fungsi berikut:

```cpp
int DUA(string S, string T) {
    int N = S.length();
    int M = T.length();

    int P = 0;

    for (int i = 0; i <= N - M; i++) {

        int Q = 1;

        for (int j = 0; j < M; j++) {
            if (S[i + j] != T[M - 1 - j]) {
                Q = 0;
            }
        }

        P += Q;
    }

    return P;
}
```

Akan dipanggil:

```cpp
DUA("BBABAABBAAABBABBAAABBBAAABBABBA", T)
```

dengan:
- panjang string `T` minimal 3.

Pertanyaannya:

> Dari seluruh kemungkinan string `T`,
> berapakah hasil kembalian terbesar
> yang mungkin dihasilkan?

Tuliskan jawaban dalam bentuk ANGKA.

:contentReference[oaicite:0]{index=0}

---

## Cara Berpikir

Fungsi:

```txt
DUA(S,T)
```

menghitung:
```txt
berapa kali reverse(T)
muncul di S
```

Karena:
```txt
panjang T minimal 3
```

maka kita cukup mencari:
```txt
substring panjang ≥ 3
yang paling sering muncul.
```

---

# Strategi Cepat

Karena:
```txt
reverse(T)
```

tetap merupakan string biasa,
maka soal berubah menjadi:

```txt
Cari substring panjang minimal 3
yang frekuensinya paling banyak.
```

# String S

```txt
BBABAABBAAABBABBAAABBBAAABBABBA
```

# Cek Substring Panjang 3

---

## "AAB"

Muncul berkali-kali:

```txt
BBAB[AAB]BAAABBABBAAABBBAAABBABBA
```

```txt
BBABAABBA[AAB]BABBAAABBBAAABBABBA
```

```txt
BBABAABBAAABBABBA[AAB]BBAAABBABBA
```

```txt
BBABAABBAAABBABBAAABBBA[AAB]BABBA
```

Total:
```txt
4 kali
```

---

## Cek Kandidat Lain

Substring lain panjang ≥ 3:
- tidak ada yang muncul lebih dari 4 kali.

Maka:
```txt
maksimum = 4
```

---

# Jawaban

4

---

## Catatan Penting OSN

Soal ini melatih:
- string matching
- substring frequency
- reverse indexing

Konsep penting:

Kode:

```cpp
T[M-1-j]
```

berarti:
```txt
fungsi sebenarnya mencari
reverse(T)
```

Jadi:
```txt
maksimum hasil
=
frekuensi substring
paling sering muncul
(dengan panjang ≥ 3).
```
