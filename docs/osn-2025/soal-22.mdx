# Soal 22 — Susunan Huruf

## Soal

Berapa banyak susunan berbeda dari huruf-huruf pada kata:

```txt
KOMPUTER
```

yang memenuhi syarat:

```txt
huruf vokal selalu berdampingan
```

Tuliskan jawaban dalam bentuk ANGKA.

---

## Cara Berpikir

Huruf pada kata:

```txt
K O M P U T E R
```

Jumlah huruf:
```txt
8
```

---

# Langkah 1 — Kelompokkan Huruf Vokal

Huruf vokal:
```txt
O, U, E
```

Karena harus selalu berdampingan,
anggap sebagai:

```txt
1 blok
```

Misalnya:

```txt
[OUE]
```

---

# Langkah 2 — Hitung Objek

Sekarang kita punya:

- 1 blok vokal
- K
- M
- P
- T
- R

Total objek:

```txt
6
```

---

# Langkah 3 — Susun Objek

## Catatan Penting OSN

Soal ini melatih:
- kombinatorika
- permutasi blok
- grouping technique

Konsep penting:

Jika beberapa objek:
```txt
harus berdampingan
```

maka:
```txt
anggap sebagai satu blok
```

Lalu:
1. susun blok
2. susun isi blok.
