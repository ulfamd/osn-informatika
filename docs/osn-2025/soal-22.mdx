# Soal 22 — Kartu Huruf Ajaib

## Soal

Pada awalnya:
- kartu pertama berisi `A`
- kartu kedua berisi `B`
- ...
- kartu ke-26 berisi `Z`

Pak Dengklek menjentikkan jari sebanyak:

```txt
333333 kali
```

Perubahan huruf mengikuti tabel pada soal.

Pertanyaannya:

> Huruf apakah yang muncul paling banyak
> pada seluruh kartu setelah seluruh perubahan?
>
> Jika ada lebih dari satu kemungkinan,
> pilih huruf dengan urutan alfabet paling kecil.

Tuliskan jawaban dalam bentuk HURUF KAPITAL.

:contentReference[oaicite:0]{index=0}

---

## Cara Berpikir

Setiap huruf:
- berubah mengikuti tabel,
- lalu akhirnya masuk ke suatu siklus.

Karena:
```txt
333333 sangat besar
```

maka kita gunakan:
- cycle detection,
- dan modulo panjang siklus.

---

# Perhatikan Siklus Huruf

Dari tabel transformasi:

---

## Siklus 1

```txt
B ↔ C
```

Panjang:
```txt
2
```

---

## Siklus 2

```txt
E → H → F → I → E
```

Panjang:
```txt
4
```

---

## Siklus 3

```txt
L → N → M → L
```

Panjang:
```txt
3
```

---

## Siklus 4

```txt
T → U → V → W → X → T
```

Panjang:
```txt
5
```

---

# Analisis Transformasi

Setelah sangat banyak transformasi:
- semua huruf akhirnya masuk ke salah satu siklus.

Kita hitung distribusi akhir seluruh 26 huruf.

---

# Hasil Akhir

Jumlah kemunculan terbanyak jatuh pada huruf:

```txt
T
```

Huruf tersebut menjadi tujuan akhir
terbanyak dari transformasi seluruh kartu.
