# Soal 35 — Pohon Keluarga Bebek

## Soal

Terdapat sebuah graph terhubung dengan:
- 12 titik
- tidak memiliki siklus

Pertanyaannya:

> Berapa banyak sisi yang dimiliki graph tersebut?

Tuliskan jawaban dalam bentuk ANGKA.

---

## Cara Berpikir

Graph:
- terhubung
- tidak memiliki siklus

disebut:

```txt
Tree (Pohon)
```

---

# Rumus Penting Tree

Jika sebuah tree memiliki:

```txt
n titik
```

maka jumlah sisi:

:contentReference[oaicite:0]{index=0}

---

# Substitusi

Diketahui:

```txt
n = 12
```

Maka:

```txt
12 - 1
=
11
```

---

# Jawaban

11

---

Soal ini melatih:


selama:
