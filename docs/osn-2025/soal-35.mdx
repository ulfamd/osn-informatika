# Soal 35 — Rahasia Petak 5x5

## Soal

Diberikan fungsi berikut:

```cpp
int RAHASIA(vector<int> A) {
    int N = A.size();
    int sum = 0;

    for (int i = 0; i < N; i++) {

        int ctr = 0;

        for (int j = 0; j < N; j++) {

            if (A[j] > A[i]) {
                ctr += 1;
            } else if (A[j] < A[i]) {
                ctr -= 1;
            }

        }

        if (ctr == 0) {
            sum += A[i];
        }

    }

    return sum;
}
```

Berapakah hasil dari:

```cpp
RAHASIA({16, 5, 20, 1, 11})
```

Tuliskan jawaban dalam bentuk ANGKA.



---

## Cara Cepat

Perhatikan variabel:

```cpp
ctr
```

- jika ada elemen lebih besar:
  ```txt
  +1
  ```

- jika ada elemen lebih kecil:
  ```txt
  -1
  ```

Agar:

```txt
ctr == 0
```

maka:
```txt
jumlah elemen lebih besar
=
jumlah elemen lebih kecil
```

---

# Artinya Apa?

Elemen tersebut adalah:

```txt
median
```

karena:
- posisinya tepat di tengah
  setelah array diurutkan.

# Urutkan Array

```txt
{16, 5, 20, 1, 11}
```

```txt

```

Elemen tengah:

```txt
11
```

yang memenuhi:
```txt
ctr == 0
```
