# Soal 26 — Modulo Tiga

## Soal

Diberikan potongan program berikut:

```cpp
int TIGA(int N) {
    if (N <= 1) {
        return 1;
    } else if (N % 3 == 0) {
        return TIGA(N - 1) + TIGA(N - 3);
    } else if (N % 3 == 1) {
        return TIGA(N - 2);
    } else {
        return TIGA(N - 3);
    }
}
```

Dari 5 pilihan berikut,
manakah yang hasil kembaliannya paling besar?

```txt
A. TIGA(2)
B. TIGA(6)
C. TIGA(14)
D. TIGA(18)
E. TIGA(25)
```

---

## Cara Berpikir

Kita hitung satu per satu.

---

# Basis Rekursi

Jika:

```txt
N <= 1
```

maka:

```txt
TIGA(N) = 1
```

---

# A. TIGA(2)

Karena:

```txt
2 % 3 = 2
```

maka:

```txt
TIGA(2)
=
TIGA(-1)
=
1
```

---

# B. TIGA(6)

Karena:

```txt
6 % 3 = 0
```

maka:

```txt
TIGA(6)
=
TIGA(5) + TIGA(3)
```

Hitung:

```txt
TIGA(5)
=
TIGA(2)
=
1
```

dan:

```txt
TIGA(3)
=
TIGA(2) + TIGA(0)
=
1 + 1
=
2
```

Jadi:

```txt
TIGA(6)
=
1 + 2
=
3
```

---

# C. TIGA(14)

Karena:

```txt
14 % 3 = 2
```

maka:

```txt
TIGA(14)
=
TIGA(11)
=
TIGA(8)
=
TIGA(5)
=
TIGA(2)
=
1
```

---

# D. TIGA(18)

```txt
TIGA(18)
=
TIGA(17) + TIGA(15)
```

Hitung:

```txt
TIGA(17)
=
TIGA(14)
=
1
```

Sekarang:

```txt
TIGA(15)
=
TIGA(14) + TIGA(12)
=
1 + TIGA(12)
```

Lalu:

```txt
TIGA(12)
=
TIGA(11) + TIGA(9)
=
1 + TIGA(9)
```

```txt
TIGA(9)
=
TIGA(8) + TIGA(6)
=
1 + 3
=
4
```

Maka:

```txt
TIGA(12)
=
1 + 4
=
5
```

```txt
TIGA(15)
=
1 + 5
=
6
```

Jadi:

```txt
TIGA(18)
=
1 + 6
=
7
```

---

# E. TIGA(25)

Karena:

```txt
25 % 3 = 1
```

maka:

```txt
TIGA(25)
=
TIGA(23)
```

```txt
23 % 3 = 2
```

maka:

```txt
TIGA(23)
=
TIGA(20)
```

```txt
20 % 3 = 2
```

maka:

```txt
TIGA(20)
=
TIGA(17)
=
1
```

Jadi:

```txt
TIGA(25) = 1
```
