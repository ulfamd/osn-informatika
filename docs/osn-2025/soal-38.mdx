# Soal 38 — Merah-Putih Nusantara

## Soal

Diberikan potongan kode berikut:

```cpp
int MERAH(int A, int B) {
    if (B == 0) {
        return A;
    } else {
        return MERAH(B, A % B);
    }
}

int PUTIH(int A, int B, int C) {
    if (C == 0) {
        return 0;
    } else if (MERAH(A, C) == B) {
        return 1 + PUTIH(A, B, C - 1);
    } else {
        return PUTIH(A, B, C - 1);
    }
}

int NUSANTARA(int A, int B) {
    return PUTIH(A, B, A);
}
```

Dari pilihan berikut,
manakah yang menghasilkan nilai terbesar?

```txt
A. MERAH(24, 4)
B. MERAH(24, 9)
C. MERAH(24, 17)
D. MERAH(24, 18)
E. MERAH(24, 34)
```

---

# Cara Cepat

Fungsi:

```cpp
MERAH(A,B)
```

adalah algoritma:

```txt
FPB / GCD
```

karena:

```cpp
MERAH(B, A % B)
```

adalah bentuk klasik:

```txt
Euclidean Algorithm
```

# Catatan Penting OSN

Soal ini melatih:
- rekursi
- FPB / GCD
- Euclidean Algorithm

Konsep penting:

```txt
FPB(a,b)
=
FPB(b, a mod b)
```

Proses terus dilakukan hingga:

```txt
b = 0
```

dan hasil akhirnya adalah:
```txt
FPB kedua bilangan.
```
