# Soal 25 — Bebek Gayung

## Soal

BENAR atau SALAH:

> Meskipun Pak Dengklek memiliki:
>
> ```txt
> N = 1000000000000000000
> ```
>
> bebek,
>
> hanya dengan dua gayung:
>
> ```txt
> 3 liter dan 100 liter
> ```
>
> banyak bebek yang tidak bisa dimandikan
> ternyata kurang dari:
>
> ```txt
> 100 ekor
> ```

Tuliskan jawaban dalam bentuk:

```txt
BENAR
```

atau

```txt
SALAH
```

## Cara Berpikir

Jumlah air yang bisa dibuat berbentuk:

:contentReference[oaicite:1]{index=1}

dengan:
- `a ≥ 0`
- `b ≥ 0`

---

# Langkah 1 — Cari FPB

Hitung:

:contentReference[oaicite:2]{index=2}

Karena FPB = 1:
- semua bilangan yang cukup besar
  pasti bisa dibentuk.

---

# Langkah 2 — Gunakan Frobenius Coin Problem

Untuk dua bilangan relatif prima:

Bilangan terbesar yang tidak bisa dibuat:

:contentReference[oaicite:3]{index=3}

Dengan:
```txt
a = 3
b = 100
```

Maka:

```txt
3×100 - 3 - 100
=
197
```

---

# Langkah 3 — Banyak Bilangan yang Tidak Bisa Dibuat

Rumus jumlah bilangan positif
yang tidak bisa dibentuk:

:contentReference[oaicite:4]{index=4}

Substitusi:

```txt
(3-1)(100-1)/2
=
2×99/2
=
99
```

---

# Kesimpulan

Jumlah bebek yang tidak bisa dimandikan:

```txt
99
```

Karena:
```txt
99 < 100
```

maka pernyataan pada soal:

```txt
BENAR
```

---

```txt
BENAR
```
