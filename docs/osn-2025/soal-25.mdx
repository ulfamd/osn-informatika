# Soal 25 — Bebek Gayung

## Soal

BENAR atau SALAH:

> Meskipun Pak Dengklek memiliki:
>
> ```txt
> N = 1000000000000000000
> ```
>
> serta dua gayung:
>
> ```txt
> X = 3 liter
> Y = 100 liter
> ```
>
> banyak bebek yang tidak bisa dimandikan ternyata kurang dari:
>
> ```txt
> 100 ekor
> ```

Tuliskan jawaban dalam bentuk:

```txt
BENAR
```

atau

```txt
SALAH
```

---

# Cara Berpikir

Jumlah air yang bisa dibuat berbentuk:

```txt
3a + 100b
```

dengan:
- a ≥ 0
- b ≥ 0

---

# Langkah 1 — Cek FPB

Karena:

```txt
FPB(3,100)=1
```

maka:
```txt
semua bilangan cukup besar
pasti bisa dibuat.
```

---

# Langkah 2 — Gunakan Rumus Frobenius

Untuk dua bilangan coprime:

```txt
a dan b
```

bilangan terbesar yang tidak bisa dibuat adalah:

```txt
ab - a - b
```

---

Substitusi:

```txt
3 × 100 - 3 - 100
=
300 - 103
=
197
```

Artinya:
- semua bilangan:
  
```txt
≥ 198
```

pasti bisa dibuat.

---

# Langkah 3 — Hitung Banyak Bilangan yang Tidak Bisa Dibuat

Jumlah bilangan positif
yang tidak bisa dibuat untuk dua bilangan coprime adalah:

```txt
(a-1)(b-1)/2
```

Substitusi:

```txt
(3-1)(100-1)/2
=
2 × 99 / 2
=
99
```

---

# Kesimpulan

Jumlah bebek yang tidak bisa dimandikan:

```txt
99
```

dan:

```txt
99 < 100
```

maka pernyataan soal:

```txt
BENAR
```

---

# Jawaban

```txt
BENAR
```

# Catatan Penting OSN

Soal ini melatih:
- teori bilangan
- linear combination
- Frobenius Coin Problem

Konsep penting:

Untuk:

```txt
ax + by
```

dengan:

```txt
FPB(a,b)=1
```

maka:
- bilangan terbesar yang tidak bisa dibuat:

```txt
ab-a-b
```

- jumlah seluruh bilangan positif
  yang tidak bisa dibuat:

```txt
(a-1)(b-1)/2
```
