# Soal 12 — Mesin BCS

## Soal

Pak Dengklek memiliki:
```txt
N = 9 koin
```

Setelah seluruh penekanan tombol selesai,
kertas hasil cetakan berisi:

```txt
543628971
```

Pertanyaannya:

> Bagaimanakah urutan penekanan tombol
> yang dilakukan oleh Pak Dengklek?
>
> Urutan harus diakhiri dengan tombol:
>
> ```txt
> S
> ```

Tuliskan jawaban dalam bentuk STRING tanpa tanda petik.


---

## Cara Berpikir

Mesin BCS bekerja seperti:

```txt
STACK (LIFO)
```

Artinya:
- koin terakhir masuk
- keluar terlebih dahulu.

Tombol:
- `B` → memasukkan koin baru
- `C` → mencetak dan mengeluarkan koin paling atas
- `S` → selesai

## Catatan Penting OSN

Soal ini melatih:
- stack simulation
- push/pop
- stack permutation

Konsep penting:

```txt
B = push
C = pop
```

Karena stack bersifat:

```txt
Last In First Out
```

maka:
- elemen terakhir masuk
- keluar terlebih dahulu.
