# Soal 23 — Susunan Huruf

## Soal

Berapa banyak susunan berbeda dari huruf-huruf pada kata:

```txt
KOMPUTER
```

yang memenuhi syarat:

```txt
huruf vokal tidak boleh berdampingan
```

Tuliskan jawaban dalam bentuk ANGKA.

---

## Cara Berpikir

Huruf pada kata:

```txt
K O M P U T E R
```

Jumlah huruf:
```txt
8
```

---

# Langkah 1 — Pisahkan Konsonan dan Vokal

## Huruf Vokal

```txt
O, U, E
```

Jumlah:
```txt
3
```

---

## Huruf Konsonan

```txt
K, M, P, T, R
```

Jumlah:
```txt
5
```

---

# Langkah 2 — Susun Konsonan

Konsonan berbeda semua.

Jumlah susunan:

:contentReference[oaicite:0]{index=0}

Hitung:

```txt
5!
=
120
```

---

# Langkah 3 — Cari Tempat Menaruh Vokal

Jika konsonan sudah disusun:

```txt
_ K _ M _ P _ T _ R _
```

Terdapat:

```txt
6 tempat
```

untuk menaruh vokal agar:
- tidak berdampingan.

---

# Langkah 4 — Pilih Tempat untuk 3 Vokal

Pilih 3 dari 6 tempat:

:contentReference[oaicite:1]{index=1}

Hitung:

```txt
20
```

---

# Langkah 5 — Susun Vokal

Huruf:
```txt
O,U,E
```

Jumlah susunan:

:contentReference[oaicite:2]{index=2}

Hitung:

```txt
6
```

---

# Langkah 6 — Total

```txt
120 × 20 × 6
=
14400
```

---

# Jawaban

14400

---

## Catatan Penting OSN

Soal ini melatih:
- kombinatorika
- gap method
- permutasi bersyarat

Konsep penting:

Jika objek:
```txt
tidak boleh berdampingan
```

maka:
1. susun objek utama dulu
2. gunakan celah/gap
3. tempatkan objek lainnya di gap tersebut.
