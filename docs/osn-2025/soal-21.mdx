# Soal 21 — Kartu Huruf Ajaib

## Soal

Pada awalnya:
- kartu pertama berisi `A`
- kartu kedua berisi `B`
- ...
- kartu ke-26 berisi `Z`

Pak Dengklek menjentikkan jari sebanyak:

```txt
3333 kali
```

Perubahan huruf mengikuti tabel pada soal.

Pertanyaannya:

> Berapa banyak huruf berbeda
> yang dapat terlihat pada seluruh kartu
> setelah seluruh perubahan dilakukan?

Tuliskan jawaban dalam bentuk ANGKA.

:contentReference[oaicite:0]{index=0}

---

## Cara Berpikir

Transformasi huruf pada tabel membentuk:

```txt
rantai dan siklus
```

Karena jumlah perubahan sangat besar:
```txt
3333
```

maka kita tidak perlu mensimulasikan satu per satu.

Kita cukup:
- mencari siklus tiap huruf,
- lalu melihat hasil akhirnya.

---

# Contoh Siklus

Beberapa huruf membentuk siklus seperti:

```txt
E → H → F → I → E
```

Panjang siklus:
```txt
4
```

Ada juga siklus:

```txt
L ↔ N ↔ M
```

dan:

```txt
T → U → V → W → X → T
```

---

# Gunakan Modulo

Karena transformasi akhirnya berulang,
kita cukup menghitung posisi akhir menggunakan:

:contentReference[oaicite:1]{index=1}

untuk setiap siklus dengan panjang:
```txt
k
```

---

# Hasil Akhir Seluruh Huruf

Setelah seluruh transformasi dilakukan,
huruf-huruf yang muncul hanyalah:

```txt
B, C,
E, F, H, I,
L, M, N,
T, U, V, W, X
```

Jumlah huruf berbeda:

```txt
14
```

---

# Jawaban

14

```txt
berulang membentuk siklus
```

```txt

```

