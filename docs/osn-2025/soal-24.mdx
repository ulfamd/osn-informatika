# Soal 24 — Bebek Gayung

## Soal

Pak Dengklek memiliki:
- `N = 1000` bebek
- dua gayung berukuran:
  - `X = 9 liter`
  - `Y = 21 liter`

Bebek ke-`P`:
- ingin mandi dengan tepat `P liter`.

Pak Dengklek boleh:
- menggunakan kedua gayung berkali-kali,
- dan setiap pengambilan air langsung digunakan.

Pertanyaannya:

> Berapa banyak bebek
> yang tidak bisa dimandikan?

Tuliskan jawaban dalam bentuk ANGKA.

:contentReference[oaicite:0]{index=0}

---

## Cara Berpikir

Jumlah air yang bisa dibuat berbentuk:

:contentReference[oaicite:1]{index=1}

dengan:
- `a ≥ 0`
- `b ≥ 0`

---

# Langkah 1 — Cari FPB

Hitung:

:contentReference[oaicite:2]{index=2}

Artinya:
- semua jumlah yang bisa dibuat
  harus kelipatan 3.

Jadi:
```txt
semua bilangan bukan kelipatan 3
pasti tidak bisa dibuat.
```

---

# Langkah 2 — Banyak Bilangan Bukan Kelipatan 3

Dari:
```txt
1 sampai 1000
```

Jumlah kelipatan 3:

:contentReference[oaicite:3]{index=3}

Maka:
- bukan kelipatan 3:

```txt
1000 - 333
=
667
```

---

# Langkah 3 — Apakah Semua Kelipatan 3 Bisa Dibuat?

Tidak.

Karena:
```txt
9a + 21b
=
3(3a + 7b)
```

Kita perlu cek:
```txt
3a + 7b
```

Bilangan kecil tertentu
masih tidak bisa dibuat.

---

# Gunakan Teorema Frobenius

Karena:
```txt
gcd(3,7)=1
```

bilangan terbesar yang tidak bisa dibuat:

:contentReference[oaicite:4]{index=4}

Artinya:
- semua bilangan di atas 11
  dapat dibuat dari:
```txt
3a+7b
```

Kalikan 3:

Bilangan terbesar kelipatan 3
yang tidak bisa dibuat:

```txt
33
```

---

# Kelipatan 3 yang Tidak Bisa Dibuat

Cek:

```txt
3   ✗
6   ✗
9   ✓
12  ✗
15  ✗
18  ✓
21  ✓
24  ✓
27  ✓
30  ✓
33  ✗
```

Yang tidak bisa:

```txt
3, 6, 12, 15, 33
```

Jumlah:
```txt
5
```

---

# Total Bebek Tidak Bisa Dimandikan

- bukan kelipatan 3:
```txt
667
```

- kelipatan 3 tetapi tetap tidak bisa:
```txt
5
```

Total:

```txt
667 + 5
=
672
```

---

# Jawaban

672

---

## Catatan Penting OSN

Soal ini melatih:
- gcd
- linear combination
- Frobenius Coin Problem

Konsep penting:

Bilangan yang dapat dibuat harus berbentuk:

:contentReference[oaicite:5]{index=5}

Jika:
```txt
gcd(x,y)=d
```

maka:
- hanya kelipatan `d`
  yang mungkin dibuat.
