# Soal 24 — Bebek Gayung

## Soal

Pak Dengklek memiliki:
- N = 1000 bebek
- dua gayung berukuran:
  
```txt
X = 9 liter
Y = 21 liter
```

Bebek ke-P hanya bisa mandi jika Pak Dengklek dapat mengambil air tepat:

```txt
P liter
```

dengan menggunakan kombinasi gayung 9 liter dan 21 liter.

Pertanyaannya:

> Berapa banyak bebek yang tidak bisa dimandikan?

Tuliskan jawaban dalam bentuk ANGKA.

---

# Cara Berpikir

Jumlah air yang bisa dibuat berbentuk:

```txt
9a + 21b
```

dengan:
- a ≥ 0
- b ≥ 0

---

# Langkah 1 — Cari FPB

Karena:

```txt
FPB(9,21) = 3
```

maka semua jumlah yang bisa dibuat:
```txt
selalu kelipatan 3
```

Artinya:
- bilangan yang bukan kelipatan 3
  pasti tidak bisa dibuat.

---

# Langkah 2 — Hitung Bilangan Bukan Kelipatan 3

Dari:

```txt
1 sampai 1000
```

Jumlah kelipatan 3:

```txt
⌊1000/3⌋ = 333
```

Maka yang bukan kelipatan 3:

```txt
1000 - 333
=
667
```

Semua ini:
```txt
tidak bisa dibuat
```

---

# Langkah 3 — Apakah Semua Kelipatan 3 Bisa?

Tidak selalu.

Karena:

```txt
9a + 21b
=
3(3a + 7b)
```

maka kita cukup melihat:

```txt
3a + 7b
```

---

# Cari Kelipatan 3 yang Tidak Bisa Dibuat

Bilangan kecil:

| Bilangan | Bisa? |
|---|---|
| 3 | Ya |
| 6 | Ya |
| 9 | Ya |
| 12 | Ya |
| 15 | Ya |
| 18 | Ya |

Karena:
```txt
3 saja sudah cukup
```

maka:
```txt
semua kelipatan 3 ≥ 3
bisa dibuat
```

---

# Jadi

Yang tidak bisa dimandikan hanyalah:
```txt
bilangan yang bukan kelipatan 3
```

Jumlahnya:

```txt
667
```

---

# Jawaban

```txt
667
```

---

# Catatan Penting OSN

Soal ini melatih:
- kombinasi linear
- FPB
- representasi bilangan

Konsep penting:

Jika:

```txt
ax + by
```

maka semua hasil:
```txt
kelipatan FPB(a,b)
```

Karena:

```txt
FPB(9,21)=3
```

maka:
```txt
hanya kelipatan 3
yang mungkin dibuat.
```
