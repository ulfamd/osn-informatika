# Soal 13 — Mesin BCS

## Soal

Pak Dengklek memiliki:
```txt
N = 6 koin
```

Dari 5 kemungkinan hasil cetakan berikut:

A.
```txt
453216
```

B.
```txt
321654
```

C.
```txt
243651
```

D.
```txt
534621
```

E.
```txt
123456
```

Pertanyaannya:

> Manakah yang sebenarnya
> tidak mungkin dihasilkan oleh Mesin BCS?


---

## Cara Berpikir

Mesin BCS bekerja seperti:

```txt
STACK (LIFO)
```

Artinya:
- elemen terakhir masuk
- keluar terlebih dahulu.

Kita cek satu per satu apakah urutan tersebut valid sebagai:

```txt
stack permutation
```

---

# A. 453216

Simulasi:

Masukkan:
```txt
1 2 3 4 5
```

Cetak:
```txt
5 4
```

Masukkan:
```txt
6
```

Cetak:
```txt
6 2 1
```

Urutan ini:
```txt
mungkin
```

---

# B. 321654

Masukkan:
```txt
1 2 3
```

Cetak:
```txt
3 2 1
```

Masukkan:
```txt
4 5 6
```

Cetak:
```txt
6 5 4
```

Urutan ini:
```txt
mungkin
```

---

# C. 243651

Masukkan:
```txt
1 2
```

Cetak:
```txt
2
```

Masukkan:
```txt
3 4
```

Cetak:
```txt
4 3
```

Masukkan:
```txt
5 6
```

Cetak:
```txt
6 5
```

Tetapi target:
```txt
243651
```

Perhatikan:
- setelah 3 keluar,
- 5 berada di bawah 6.

Agar:
```txt
5 keluar sebelum 1
```
masih mungkin.

Lanjut:
- akhirnya bisa dibentuk.

Urutan ini:
```txt
mungkin
```

---

# D. 534621

Cetak:
```txt
5
```
berarti harus memasukkan:
```txt
1 2 3 4 5
```

Stack tersisa:
```txt
1 2 3 4
```

Cetak berikutnya:
```txt
3
```

Masalah:
```txt
4 berada di atas 3
```

Maka:
```txt
3 tidak bisa keluar sebelum 4
```

Karena stack bersifat:
```txt
LIFO
```

Jadi urutan ini:
```txt
tidak mungkin
```

---

# E. 123456

Masukkan lalu langsung cetak satu per satu.

Jelas:
```txt
mungkin
```

---

# Jawaban

```txt
D
```

## Catatan Penting OSN

Soal ini melatih:
- stack permutation
- simulasi stack
- validasi urutan output

Konsep penting:

Jika:
```txt
x berada di bawah y
```

maka:
```txt
x tidak bisa keluar sebelum y keluar.
```

Karena:
- stack menggunakan prinsip:

```txt
Last In First Out
```
