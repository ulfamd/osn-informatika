# Soal 31 — FUN dan GSI

## Soal

Berapakah hasil kembalian dari:

```cpp
FUN({3, -2, 4, -3, 1, -4, 2, -1, 4, 3, -2, 3, -4, 2, -2, 3})
```

Tuliskan jawaban dalam bentuk ANGKA.


---

## Inti Soal

Fungsi:

```cpp
FUN(A)
```

sebenarnya mencari:

```txt
jumlah subarray terbesar
```

karena:

```cpp
out = max(out, GSI(B, i, j))
```

---

# Cara Cepat (Kadane)

Tidak perlu cek semua kombinasi.

Cukup:
- simpan `sum sekarang`
- simpan `maksimum`

Jika:
```txt
sum menjadi negatif
```

maka:
```txt
reset
```

karena:
- prefix negatif
  hanya memperburuk hasil berikutnya.

---

# Hitung Cepat

Array:

```txt
3, -2, 4, -3, 1, -4, 2, -1,
4, 3, -2, 3, -4, 2, -2, 3
```

---

| Angka | Sum Sekarang | Maksimum |
|---|---|---|
| 3 | 3 | 3 |
| -2 | 1 | 3 |
| 4 | 5 | 5 |
| -3 | 2 | 5 |
| 1 | 3 | 5 |
| -4 | -1 | 5 |

Karena negatif:
```txt
reset
```

---

Lanjut:

| Angka | Sum Sekarang | Maksimum |
|---|---|---|
| 2 | 2 | 5 |
| -1 | 1 | 5 |
| 4 | 5 | 5 |
| 3 | 8 | 8 |
| -2 | 6 | 8 |
| 3 | 9 | 9 |
| -4 | 5 | 9 |
| 2 | 7 | 9 |
| -2 | 5 | 9 |
| 3 | 8 | 9 |

---

# Nilai Terbesar

```txt
9
```

```txt
9
```

## Trik Cepat OSN

Jika soal:
```txt
mencari jumlah subarray maksimum
```

langsung pikir:

```txt
Kadane Algorithm
```

Pola inti:

```txt
sum += A[i]

jika sum < 0:
    reset
```

Kompleksitas:
```txt
O(N)
```

sangat cepat untuk OSN.
