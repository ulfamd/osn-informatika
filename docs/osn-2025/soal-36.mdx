# Soal 36 — Rahasia Petak 5x5

## Soal

Diberikan fungsi berikut:

```cpp
int RAHASIA(vector<int> A) {
    int N = A.size();
    int sum = 0;

    for (int i = 0; i < N; i++) {

        int ctr = 0;

        for (int j = 0; j < N; j++) {

            if (A[j] > A[i]) {
                ctr += 1;
            } else if (A[j] < A[i]) {
                ctr -= 1;
            }

        }

        if (ctr == 0) {
            sum += A[i];
        }

    }

    return sum;
}

int PETAK(vector<vector<int>> B) {
    int N = B.size();

    vector<int> C;

    for (int i = 0; i < N; i++) {
        C.push_back(RAHASIA(B[i]));
    }

    return RAHASIA(C);
}
```

Berapakah hasil dari:

```cpp
PETAK(
 {16, 5, 20, 1, 11},
 {2, 17, 13, 14, 19},
 {15, 7, 21, 18, 9},
 {25, 6, 4, 12, 22},
 {23, 3, 24, 8, 10}
)
```

Tuliskan jawaban dalam bentuk ANGKA.


---

## Cara Cepat

Dari soal sebelumnya:

```txt
RAHASIA(A)
```

menghasilkan:

```txt
median dari array A
```

karena:
- elemen dipilih saat
  jumlah elemen lebih besar
  = jumlah elemen lebih kecil.
