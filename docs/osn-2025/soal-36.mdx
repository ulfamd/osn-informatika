# Soal 36 — Pohon Keluarga Bebek

## Soal

Terdapat sebuah tree dengan:
- 15 titik

Pertanyaannya:

> Berapa jumlah derajat seluruh titik pada tree tersebut?

Tuliskan jawaban dalam bentuk ANGKA.

---

## Cara Berpikir

Pada graph tak berarah berlaku:

```txt
jumlah seluruh derajat
=
2 × jumlah sisi
```

Ini disebut:

```txt
Handshaking Lemma
```

---

# Langkah 1 — Cari Jumlah Sisi Tree

Untuk tree:

:contentReference[oaicite:0]{index=0}

Dengan:
```txt
n = 15
```

Maka:

```txt
15 - 1
=
14
```

---

# Langkah 2 — Hitung Jumlah Derajat

Gunakan rumus:

:contentReference[oaicite:1]{index=1}

Dengan:
```txt
E = 14
```

Maka:

```txt
2 × 14
=
28
```

---

# Jawaban

28
