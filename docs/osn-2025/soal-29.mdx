# Soal 29 — FUN dan GSI

## Soal

Diberikan potongan program berikut:

```cpp
int GSI(vector<int> B, int x, int y) {
    return B[y] - B[x - 1];
}

int FUN(vector<int> A) {
    int N = A.size();

    vector<int> B;
    B.push_back(0);

    for (int i = 0; i < N; i++) {
        B.push_back(B[i] + A[i]);
    }

    int out = 0;

    for (int i = 1; i <= N; i++) {
        for (int j = i; j <= N; j++) {
            out = max(out, GSI(B, i, j));
        }
    }

    return out;
}
```

Apabila:

```txt
A = {1, 2, 3, 4, 5}
```

maka dari 5 pilihan berikut,
manakah isi vector `B`?

```txt
A. {1, 3, 6, 10, 15}
B. {0, 1, 3, 6, 10, 15}
C. {0, 1, 2, 3, 4, 5}
D. {0, 2, 4, 6, 8, 10}
E. {15, 15, 15, 15, 15}
```

## Cara Berpikir

Vector `B` dibentuk menggunakan:

```cpp
B.push_back(B[i] + A[i]);
```

Artinya:
- setiap elemen `B`
  menyimpan jumlah prefix.

# Langkah Perhitungan

Awalnya:

```txt
B = {0}
```

---

## i = 0

```txt
B[0] + A[0]
=
0 + 1
=
1
```

Maka:

```txt
B = {0,1}
```

---

## i = 1

```txt
B[1] + A[1]
=
1 + 2
=
3
```

Maka:

```txt
B = {0,1,3}
```

---

## i = 2

```txt
3 + 3 = 6
```

```txt
B = {0,1,3,6}
```

---

## i = 3

```txt
6 + 4 = 10
```

```txt
B = {0,1,3,6,10}
```

---

## i = 4

```txt
10 + 5 = 15
```

```txt
B = {0,1,3,6,10,15}
```
