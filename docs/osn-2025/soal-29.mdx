# Soal 29 — FUN dan GSI

## Soal

Diberikan potongan kode berikut:

```cpp
int GSI(vector<int> B, int x, int y) {
    return B[y] - B[x - 1];
}

int FUN(vector<int> A) {
    int N = A.size();

    vector<int> B;
    B.push_back(0);

    for (int i = 0; i < N; i++) {
        B.push_back(B[i] + A[i]);
    }

    int out = 0;

    for (int i = 1; i <= N; i++) {
        for (int j = i; j <= N; j++) {
            out = max(out, GSI(B, i, j));
        }
    }

    return out;
}
```

Jika:

```txt
A = {1, 2, 3, 4, 5}
```

maka manakah isi vector:

```txt
B
```

# Cara Cepat

Perhatikan bagian:

```cpp
B.push_back(B[i] + A[i]);
```

Artinya:
- setiap elemen baru pada `B`
  adalah:
  
```txt
jumlah sebelumnya + A[i]
```

Ini disebut:

```txt
prefix sum
```
