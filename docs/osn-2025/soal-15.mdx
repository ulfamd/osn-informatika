# Soal 15 — Perjalanan ke Timur

## Soal

Bebek-bebek Pak Dengklek ingin pergi:
- dari titik A
- menuju titik H

Mereka hanya boleh:
```txt
bergerak ke arah kanan
```

Pada graph terdapat:
- 3 jalur rahasia bergembok,
- dan pada Soal 15,
  bebek memiliki:
  
```txt
3 kunci sakti
```

Karena:
- ada tepat 3 gembok,
- dan tersedia 3 kunci,

maka:
```txt
seluruh jalur dapat digunakan.
```

Pertanyaannya:

> Berapa banyak kemungkinan rute berbeda
> dari A ke H?

Tuliskan jawaban dalam bentuk ANGKA.



---

## Cara Berpikir

Karena semua jalur terbuka,
kita hitung jumlah rute menggunakan:

```txt
Dynamic Programming pada DAG
```

---

# Dari A ke B

Ada:
```txt
2 jalur
```

Maka:

```txt
B = 2
```

---

# Dari B ke C

Ada:
```txt
1 jalur
```

Maka:

```txt
C = 2
```

---

# Dari B ke D

Ada:
```txt
2 jalur
```

Maka:

```txt
D = 2 × 2
=
4
```

---

# Dari B ke E

Ada:
```txt
1 jalur
```

Maka:

```txt
E = 2
```

---

# Menuju F

Sekarang:
- jalur D → F terbuka
- jalur E → F ada 2

Maka:

Dari D:
```txt
4 × 1 = 4
```

Dari E:
```txt
2 × 2 = 4
```

Total:

```txt
F = 8
```

---

# Menuju G

Jalur menuju G:
- C → G : 1 jalur
- D → G : 2 jalur
- F → G : 1 jalur

Hitung:

Dari C:
```txt
2 × 1 = 2
```

Dari D:
```txt
4 × 2 = 8
```

Dari F:
```txt
8 × 1 = 8
```

Total:

```txt
G = 18
```

---

# Dari G ke H

Ada:
```txt
2 jalur
```

Maka:

```txt
H = 18 × 2
=
36
```

---

# Jawaban

36

---

## Catatan Penting OSN

Soal ini melatih:
- graph DAG
- counting paths
- dynamic programming

Konsep penting:

Pada DAG:

```txt
jumlah cara ke node
=
jumlah seluruh cara
dari predecessor.
```

Karena:
- semua jalur sekarang terbuka,
- jumlah rute bertambah signifikan.
