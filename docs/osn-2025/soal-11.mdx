# Soal 11 — Mesin BCS

## Soal

Pak Dengklek memiliki sebuah mesin bernama:

```txt
Mesin BCS
```

Mesin memiliki 3 tombol:

- `B` (Buka)
- `C` (Cetak)
- `S` (Stop)

Aturan mesin:

---

## Tombol B

Ketika ditekan:
- bagian atas tabung terbuka,
- lalu dimasukkan koin dengan nomor terkecil
  yang belum pernah dimasukkan sebelumnya.

---

## Tombol C

Ketika ditekan:
- mesin mencetak nomor koin paling atas,
- lalu mengeluarkan koin tersebut.

Jika tabung kosong:
```txt
error
```

---

## Tombol S

Ketika ditekan:
- tabung harus kosong,
- lalu mesin selesai.

---

Diketahui:
- terdapat `N = 4` koin,
- urutan tombol:

```txt
BBCBBCCCS
```

Pertanyaannya:

> Berapakah hasil cetakan pada kertas?

Tuliskan jawaban tanpa spasi.

---

## Cara Berpikir

Mesin bekerja seperti:

```txt
STACK
```

atau:
```txt
LIFO
```

(last in first out)

---

# Simulasi Langkah demi Langkah

Awalnya:
```txt
stack kosong
```

---

```txt
1

```

```txt
[1,2]
```

```txt
2
```

## Catatan Penting OSN

Soal ini melatih:
- stack
- simulasi algoritma
- LIFO structure

Konsep penting:

Stack bekerja dengan aturan:

```txt
Last In First Out
```

Koin terakhir masuk:
- keluar lebih dulu.
