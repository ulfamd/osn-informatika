# OSN Informatika SMA

## Apa Itu OSN?

OSN adalah singkatan dari:

```txt
Olimpiade Sains Nasional
```

OSN merupakan kompetisi akademik tingkat nasional untuk siswa SMA/sederajat di Indonesia yang diselenggarakan secara resmi oleh pemerintah.

Tujuan OSN:
- melatih kemampuan berpikir tingkat tinggi
- meningkatkan kemampuan problem solving
- melatih logika dan analisis
- mempersiapkan siswa menuju kompetisi internasional

Bidang OSN sangat banyak, salah satunya adalah:

```txt
OSN Informatika
```

---

# Apa Itu OSN Informatika?

OSN Informatika adalah kompetisi yang berfokus pada:
- logika
- algoritma
- pemrograman
- matematika diskrit
- computational thinking

Berbeda dengan belajar coding biasa,
OSN Informatika lebih fokus pada:

```txt
cara berpikir
```

daripada sekadar menghafal syntax.

# Tips Belajar OSN Informatika

## 1. Fokus pada Logika

OSN bukan lomba hafalan syntax.

Yang paling penting:
```txt
cara berpikir
```

---

## 2. Biasakan Analisis Soal

Saat membaca soal:
- cari pola
- cari constraint
- cari kemungkinan solusi tercepat

---

## 3. Pelajari dari Soal Tahun Sebelumnya

Ini sangat penting.

Karena pola soal OSN:
- sering berulang
- konsepnya mirip
- tingkat kesulitannya bertahap

---

## 4. Jangan Langsung Coding

Biasakan:
1. pahami soal
2. cari pola
3. buat strategi
4. baru coding

---

## 5. Latihan Sedikit Tapi Konsisten

Lebih baik:
```txt
1 soal per hari
```

daripada:
```txt
20 soal lalu berhenti lama
```

# Trik dan Strategi Mengerjakan Soal OSN

## Cari Pola

Contoh:
- genap ganjil
- modulo
- simetri
- pola berulang

---

## Cari Kasus Kecil Dulu

Misalnya:
```txt
N = 1
N = 2
N = 3
```

---

## Gunakan Eliminasi

Kadang jawaban pilihan ganda bisa diselesaikan tanpa menghitung semua.

```txt
Apakah solusi ini cukup cepat?
```
